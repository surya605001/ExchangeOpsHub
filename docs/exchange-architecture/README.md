# Exchange Architecture

---

## Module Overview

The **Exchange Architecture** module explains the internal design and components of Microsoft Exchange Server.

Understanding Exchange architecture is essential before administering, troubleshooting, upgrading, or designing an Exchange environment.

This module builds the foundation required to understand how Exchange components interact with Active Directory, Windows Server, IIS, networking, and client applications.

---

# Learning Objectives

After completing this module, you will be able to:

- Understand Exchange Server architecture
- Identify Exchange components
- Explain the mail flow pipeline
- Understand Exchange services
- Explain client connectivity
- Understand mailbox databases
- Explain high availability concepts
- Understand Exchange certificates
- Understand Exchange virtual directories
- Troubleshoot architecture-related issues

---

# Prerequisites

Before starting this module, you should complete:

- System Study
- Active Directory Discovery
- Server Inventory
- Database Inventory
